About the BCA Apps

This suite of applications has been developed by the students of BCA with the goal of making all of our lives more efficient. If you have ideas for new applications, questions, or feedback please contact Bryan Respass.

This page provides the history of the development of the applications.

June 2018

Transcripts (Beta launch!)

Features

  • Students can submit non-college transcript requests.
  • Counseling office can track and record status on all requests.

  • Nahyun (Kate) Kim
  • Kairui Huang

January 2017

Three new BCA apps have been released since Career Day was created last year. In this release, Career Day has been updated to include new features and styles from those applications while also improving its administrative pages.

The official release of the teacher proctoring application.

Career Day Features

  • Color theme and styling redesign, imported from work on other BCA applications.
  • Full set of administration pages; enabling Ms. Pinke to run all aspects of the application.

Teacher Proctoring Features

  • Teacher page updated to list up to 12 tests without scrolling.
  • New fields for tests added, including Number of Students, Lead Proctor, and Active.
  • Dates now include day of week wherever listed for teachers.
  • Cleaned up the administrative test listing page.
  • PDF report listing teachers registered to proctor a given test.

Development Team

  • Teacher Proctoring: Celina Peralta James Lepone David Yi Jefferson Xu
